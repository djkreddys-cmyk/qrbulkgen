# QRBulkGen Route and Screen Map - March 13, 2026

| Route | Purpose | Auth |
|---|---|---|
| `/` | Main landing page | No |
| `/pricing` | Pricing and feature comparison | No |
| `/blog` | Blog index | No |
| `/blog/[slug]` | Blog article page | No |
| `/login` | User login | No |
| `/register` | User registration | No |
| `/generate/single` | Single QR generator | Yes |
| `/generate/bulk` | Bulk QR generator | Yes |
| `/dashboard` | User dashboard and recent jobs | Yes |

## Navigation Notes
- Landing page CTA should route to `/generate/bulk` after login.
- Pricing CTA should route to `/register` or authenticated generation pages.
- Blog posts should link to pricing and generator routes.
- Mobile app does not include blog, pricing, or SEO landing pages.
